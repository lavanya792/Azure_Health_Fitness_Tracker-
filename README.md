AI-Enabled Health & Nutrition Tracker

A lightweight, Apple-style Streamlit web app that helps users track daily meals, calories, nutrients, steps, and receive intelligent suggestions.
It integrates Azure Cognitive Services for Vision, Sentiment Analysis, Key Phrase Extraction, and Translation to enhance the health-tracking experience.
